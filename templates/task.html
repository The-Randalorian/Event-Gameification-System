Instructions

Bellow you will see the list of your current tasks and completed tasks. The list is real time so as you complete tasks you will be able to watch the list update to reflect this. Hints are directly linked to each task, so each hint next to the task will relate only to that task. These hints will also move to the completed sections once a task is completed.

To start a task, simply click on the given task to be taken to that tasks page. Once you are on the tasks page you will see a button prompting you to scan a QR code. Once you have found the proper QR code, click the button and scan the QR code with the given window. This will now mark this task as complete and it will automatically be moved into the completed section.
